# Group-02
Expense Tracker System:
A web-based system that connects with bank accounts to track income and expenses of individual users(students, working professionals) over time. It categorises spending and sends warnings when the user exceeds their budget.

Lost & Found System:
A platform where users can upload information about lost items. Other users can view posts, comment, and help notify the owner if the item is found.

Productivity website:
A student-focused website that includes note-taking, a study timer (Pomodoro), study music, and interactive features like challenges and quizzes to improve focus and learning.
